# Run a StarburstFuture

Submits the future task to AWS for execution (EC2 by default, or
Fargate)

## Usage

``` r
# S3 method for class 'StarburstFuture'
run(future, ...)
```

## Arguments

- future:

  A StarburstFuture object

- ...:

  Additional arguments

## Value

The future object (invisibly)
