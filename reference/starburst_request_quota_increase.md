# Request quota increase (user-facing)

Request quota increase (user-facing)

## Usage

``` r
starburst_request_quota_increase(vcpus = 500, region = NULL)
```

## Arguments

- vcpus:

  Desired vCPU quota

- region:

  AWS region (default: from config)

## Examples

``` r
if (FALSE) { # \dontrun{
starburst_request_quota_increase(vcpus = 500)
} # }
```
