# Rebuild environment image

Rebuild environment image

## Usage

``` r
starburst_rebuild_environment(region = NULL, force = FALSE)
```

## Arguments

- region:

  AWS region (default: from config)

- force:

  Force rebuild even if current environment hasn't changed

## Examples

``` r
if (FALSE) { # \dontrun{
starburst_rebuild_environment()
} # }
```
